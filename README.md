# She Can Foundation Contact Form

A simple, responsive, and interactive mini project built using **HTML, CSS, and JavaScript**.

## Project Overview

This project was created for a Full Stack Development Internship Task. The webpage contains a contact form for **She Can Foundation** with the required fields:

- Name
- Email
- Message
- Submit button

After successful submission, the form shows a success message: **Form Submitted Successfully**.

## Features

### Required Features
- Name field
- Email field
- Message field
- Submit button
- Success message after submission

### Additional Features Added
- Responsive design for mobile and desktop
- Light/Dark theme toggle
- Live form validation
- Error message display only after user interaction
- Character counter for message field
- Progress bar showing form completion
- Clear form button
- Last submitted data preview
- Clean and modern UI
